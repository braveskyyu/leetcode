关键是为什么要剪掉一(即:n-1)，还是不太理解，只能用推算发现确实要剪掉一。

```java
class Solution {
    public int findNthDigit(int n) {
        if (n<10) return n;
        int d = 1;
        long m=9;
        int start=1;
        while(n>m*d)
        {
            n-=m*d;
            m*=10;
            start*=10;
            d++;
        }
        
        start+=(n-1)/d;
        return Character.getNumericValue(String.valueOf(start).charAt((n-1)%d));
    }
}
```
