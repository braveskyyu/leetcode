经典的双指针问题,一个指针用来loop数组，另一个指针用来记载不是0的元素下标

```java
class Solution {
    public void moveZeroes(int[] nums) {
        int m=0;
        for (int i=0;i<nums.length;i++)
        {
            if (nums[i]!=0)
            {
                nums[m]=nums[i];
                if (m!=i) nums[i]=0;
                m++;
            }
        }
    }
}
```
