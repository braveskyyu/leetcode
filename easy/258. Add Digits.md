这题没什么难的，只是注意c-'0'的这种写法，如果c是int的话，返回的是c代表的int值。

```java
class Solution {
    public int addDigits(int num) 
    {
        if (num<10) return num;
        String s = String.valueOf(num);
        char[] cArr = s.toCharArray();
        num=0;
        for (char c: cArr)
        {
            num+=(int)(c-'0');
        }
        return addDigits(num);
    }
}
```

class Solution {
    public int addDigits(int n) {
        if (n==0) return 0;
        if (n%9==0) return 9;
        return n%9;
    }
}
